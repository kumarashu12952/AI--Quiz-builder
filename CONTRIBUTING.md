# Contributing to AI Quiz Builder

We'd love your contributions! Here's how to get started:

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/AI--Quiz-builder.git`
3. Create a feature branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Commit: `git commit -am 'Add some feature'`
6. Push: `git push origin feature/your-feature-name`
7. Submit a pull request

## Development Setup

```bash
# Backend
cd backend
npm install
npm run dev

# Frontend (in another terminal)
cd frontend
npm install
npm run dev
```

## Code Style

- Use ESLint for JavaScript
- Follow the existing code structure
- Write meaningful commit messages
- Add comments for complex logic

## Testing

```bash
# Run tests
npm test

# Run with coverage
npm test -- --coverage
```

## Reporting Issues

1. Check existing issues first
2. Provide detailed description
3. Include steps to reproduce
4. Share relevant error messages

## Feature Requests

Open an issue with:
- Clear description of the feature
- Why it would be useful
- Any relevant examples

## Pull Request Process

1. Update documentation as needed
2. Add tests for new features
3. Ensure all tests pass
4. Request review from maintainers
5. Address feedback
6. Merge when approved

## Code of Conduct

Be respectful and constructive in all interactions.
